# Student Grade Tracker

This is a Java console-based application developed as part of the 
CodeAlpha Java Programming Internship.

## Description
The Student Grade Tracker application allows users to enter the marks 
of multiple students and calculates the average, highest, and lowest marks.
It then displays a summary report.

## Features
- Accepts student marks from the user
- Calculates average marks
- Finds highest and lowest marks
- Displays a summary report

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Arrays
- Scanner class

## How to Run
1. Compile the program using:
   javac StudentGradeTracker.java
2. Run the program using:
   java StudentGradeTracker
